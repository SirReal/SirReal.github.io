Esta página

Esta página está alojada con GitHub pages. GitHub es bastante fantástico. Y, como no, utilizo git, el sistema de control de versiones.

La página está diseñada con Sass con Compass. Son herramientas que facilitan el trabajo de desarrollo de páginas web, en especial el trabajo de CSS (hojas de estilo). Habitualmente trabajo con LESS y Bootstrap, que podrían llamarse alternativas a Sass y Compass, pero he querído probar algo nuevo para esto. Ya te contaré…

GitHub pages emplean Jekyll, y he decidido emplearlo también. Es una herramienta que facilita la generación de páginas web estáticas (HTML) a partir de plantillas. De momento me gusta. Normalmente trabajaría con Wordpress, pero para un proyecto pequeño y relativamente sencillo como éste, wordpress es demasiado.

Otra herramienta que utilizo para esta página es Grunt. Me ahorra tener que hacer cosas cotidianas y dolores de cabeza.

Todo el código de esta página está disponible en el repositorio.
